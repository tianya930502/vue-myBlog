### 安装

---

``` bash
# 开发环境运行
$ git clone https://github.com/tianya930502/vue-myBlog.git
$ cd vue-myBlog
$ yarn 或者 npm install
$ yarn run dev

浏览器中打开localhost:3000

# 生产环境运行
$ yarn run build
$ yarn start

$ yarn run generate
```
